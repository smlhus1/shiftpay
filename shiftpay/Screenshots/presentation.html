AI-powered OCR

Claude Vision reads your timesheet

Snap a photo of any printed or handwritten schedule. Claude Haiku 4.5 Vision extracts every shift — dates, times, and type.

Works with Norwegian healthcare schedules out of the box. Messy formats, abbreviations, handwriting — all handled.

The image is processed once and immediately discarded. Nothing is stored.

Your shift hub

Did you complete the shift?

Overtime. Enter extra hours and minutes.

Now you have proof — not just a schedule, but what actually happened.

How we built it

Vibe coding with real engineering

Two AI-powered editors. One human making the decisions.

Claude Code
CLI — architecture, pipelines, deep work
/sec-eval — Full 5-phase security audit pipeline. Spawns specialized agents (security-auditor, dep-config-scanner, attack-scenario-builder) sequentially, each building on the previous phase. Identified 14 findings, fixed all in one session.
/refactor — Structured code refactoring with analyze → plan → implement → verify cycle. Used for extracting date utilities, fixing type errors, and cleaning up routing.
/design-review — Design evaluation with accessibility audit agent. Found 28 UU findings across 4 severity levels with concrete WCAG references and fixes.
/commit — Smart commits: scans for debug code, secrets, and lint issues before staging. Generates conventional commit messages from the actual diff.
